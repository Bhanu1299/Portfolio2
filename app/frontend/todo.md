# Personal Portfolio - CS Master's Graduate

## Design Guidelines

### Design References
- **Brittany Chiang** (brittanychiang.com): Clean developer portfolio, dark theme
- **Minimal & Bold**: Dark mode with vibrant accent colors, glassmorphism effects
- **Style**: Dark Futuristic + Glassmorphism + Developer-focused

### Color Palette
- Primary Background: #0A0A0F (Deep Dark)
- Secondary Background: #12121A (Card surfaces)
- Accent Primary: #6366F1 (Indigo - links, highlights)
- Accent Secondary: #8B5CF6 (Violet - gradients)
- Accent Tertiary: #06B6D4 (Cyan - hover states)
- Text Primary: #F8FAFC (White)
- Text Secondary: #94A3B8 (Slate gray)
- Glass Border: rgba(255, 255, 255, 0.08)

### Typography
- Headings: Inter, font-weight 700-800
- Body: Inter, font-weight 400
- Code/Tech: JetBrains Mono or monospace

### Key Component Styles
- **Cards**: Glassmorphism with backdrop-blur, subtle border glow on hover
- **Buttons**: Gradient indigo-to-violet, rounded-full, glow shadow on hover
- **Badges**: Semi-transparent bg with colored text, rounded-full
- **Navigation**: Fixed glassmorphism bar with blur backdrop
- **Animations**: Fade-in on scroll, typing effect, subtle float animations
